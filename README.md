MarsXZ Media

MarsXZ Media — настольное приложение для загрузки видео и аудио с YouTube.

Возможности:
• Загрузка видео и аудио
• Извлечение только аудио (MP3)
• Объединение потоков через ffmpeg
• Простое и лёгкое приложение для слабых ПК
• Работает без установленного .NET (self-contained build)

Используемые инструменты:
- yt-dlp — загрузка медиа
- ffmpeg — обработка и конвертация
- node — внутренние компоненты
